  <action>
Create `src/lib/utils/motion.ts` with reduced motion detection.

**Export these utilities:**

1. `prefersReducedMotion(): boolean`
   - Returns true if user has prefers-reduced-motion: reduce
   - Returns false on server (fail-open for SSR)
   - Uses window.matchMedia for detection

2. `getAnimationDuration(baseDuration: number): number`
   - Returns 0 if reduced motion is preferred
   - Returns baseDuration otherwise
   - Useful for conditional animation timing

3. `getMotionSafeProps(animationProps: object): object`
   - Returns animationProps if motion allowed
   - Returns version with duration: 0 if motion reduced
   - Useful for passing to GSAP

**Pattern:**
```typescript
import { browser } from '$app/environment';

export function prefersReducedMotion(): boolean {
  if (!browser) return false;
  return window.matchMedia('(prefers-reduced-motion: reduce)').matches;
}
```

**Interface:**
```typescript
interface AnimateOptions extends gsap.TweenVars {
  type?: 'to' | 'from' | 'fromTo' | 'set';
  scrollTrigger?: ScrollTrigger.Vars;
  fromVars?: gsap.TweenVars; // For fromTo
}
```

**Key patterns:**
1. Lazy load GSAP with dynamic import
2. Store tween reference for cleanup
3. In destroy(), call `tween.kill()` and `scrollTrigger.kill()` if exists
4. Check reduced motion - if true, set duration: 0

**Usage example (for documentation):**
```svelte

  type: 'from',
  opacity: 0,
  y: 30,
  duration: 0.6
}}>

```
