<task type="auto">
  <name>Task 2: Create design tokens file</name>
  <files>src/lib/styles/tokens.css</files>
  <action>
Create `src/lib/styles/tokens.css` with all design tokens using Tailwind CSS 4's `@theme` directive.

**Required tokens (from RESEARCH.md):**

**Colors (Semantic naming, NO mint/magenta):**
- `--color-surface`: #f5f0e8 (cream/parchment background)
- `--color-surface-warm`: #f0ebe0 (warmer variant)
- `--color-text`: #1a1a1a (black ink)
- `--color-text-muted`: #4a4a4a (secondary text)
- `--color-primary`: #c55a3d (terracotta accent)
- `--color-primary-dark`: #a84830 (terracotta hover)
- `--color-botanical`: #5a7c65 (muted green for botanical elements ONLY)
- `--color-bone`: #e8e0d4 (light stone tone)
- `--color-burgundy`: #722f37 (deep red accent)

**Typography (Golden Ratio Scale 1.618, base 18px):**
- Font families: `--font-serif` (Playfair Display), `--font-sans` (Source Sans 3)
- Scale: --text-xs through --text-6xl
- Line heights: generous body (1.8), tighter for headings

**Spacing:**
- `--spacing-section`: 10rem (160px between sections)
- `--spacing-section-sm`: 7.5rem (120px mobile)

**Grid:**
- `--content-max`: 56.25rem (900px narrow editorial)
- `--content-wide`: 75rem (1200px when needed)

**Easing (Organic/botanical feel):**
- `--ease-organic`: cubic-bezier(0.22, 1, 0.36, 1) - out-quint
- `--ease-reveal`: cubic-bezier(0.16, 1, 0.3, 1) - out-expo dramatic
- `--ease-subtle`: cubic-bezier(0.4, 0, 0.2, 1) - UI interactions

See RESEARCH.md Pattern 1 for exact implementation.
  </action>
  <verify>
File exists at `src/lib/styles/tokens.css` and contains `@theme {` block with all specified tokens.
  </verify>
  <done>
tokens.css defines complete design system: colors, typography scale, spacing, grid constraints, and easing curves.
  </done>
</task>
