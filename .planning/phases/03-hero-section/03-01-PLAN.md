  <name>Task 2: Create HeroIllustration.svelte with abstract organic linework</name>
  <files>src/lib/components/hero/HeroIllustration.svelte</files>
  <action>
Create `src/lib/components/hero/HeroIllustration.svelte` — a pure inline SVG component with abstract organic linework.

**Create the directory first:** `src/lib/components/hero/`

**Design direction (from CONTEXT.md locked decisions):**
- Abstract organic linework — flowing lines suggesting growth, roots, neural networks (NOT a literal plant or figure)
- "Medical textbook meets art book" — scholarly authority, Nigrum Corpus aesthetic
- Subtle background presence — noticeable but restrained, doesn't compete with typography
- Muted terracotta (#c55a3d) or botanical green (#5a7c65) in select areas on cream

**SVG specification:**
- viewBox: "0 0 1200 900" (landscape, fills hero viewport)
- All animatable elements use `<path>` (NOT `<rect>`, `<circle>`, or `<ellipse>` — avoid iOS Safari DrawSVG bug per research Pitfall 3)
- Every animatable path gets `class="hero-path"` for GSAP targeting
- Every path MUST have `stroke` and `stroke-width` attributes set (DrawSVG requires stroke per Pitfall 1)
- Use `fill="none"` on the SVG root. Selected paths may have subtle `fill` with low opacity for terracotta/green accents
- NO CSS animations — no `@keyframes`, no `animation`, no `transition` in `<style>` block
- NO JavaScript — purely structural SVG markup

**Composition (~20-25 paths for desktop):**
Design the SVG with these conceptual layers:

1. **Root/growth network (8-10 paths):** Branching lines that flow from a central area outward, suggesting root systems or neural dendrites. Use organic bezier curves (`C` and `S` path commands), not straight lines. Varying stroke widths (0.5 to 1.5). Primary color: `stroke="#1a1a1a"` with `opacity="0.15"` to `opacity="0.3"`.

2. **Circular geometry frame (3-4 paths):** Large circular arcs (using `<path>` with arc commands, NOT `<circle>`) suggesting the Vitruvian circle and scholarly framework. Thin strokes (0.5-0.8). Color: `stroke="#c55a3d"` with `opacity="0.2"`.

3. **Flowing organic lines (6-8 paths):** Sweeping curves suggesting river courses, meridian lines, or growth trajectories. These should flow through and around the composition creating visual rhythm. Varying strokes (0.8-1.2). Mix of `stroke="#1a1a1a"` and `stroke="#5a7c65"` at low opacity.

4. **Accent fills (2-3 closed paths):** Small organic shapes with subtle terracotta fill (`fill="#c55a3d"` with `fill-opacity="0.08"`) suggesting highlighted anatomical regions or growth nodes. These add warmth without competing with text.

**Mobile simplification:**
Add a CSS class structure that allows the parent to hide certain paths on mobile:
- Wrap detail paths in a `<g class="hero-detail">` group (the paths that get hidden on mobile)
- Core paths (~8-12) remain outside this group and are always visible
- The parent component handles responsive visibility via CSS (not this component's concern)

**Component props:**
```svelte
<script lang="ts">
  interface Props {
    class?: string;
  }
  let { class: className = '' }: Props = $props();
</script>
```

Pass `className` to the root SVG element's `class` attribute.

**Important anti-patterns to avoid:**
- Do NOT add CSS `stroke-dasharray` or `stroke-dashoffset` (GSAP DrawSVG handles this)
- Do NOT set `opacity: 0` on paths in CSS (GSAP handles initial state)
- Do NOT import any JavaScript libraries
- Do NOT use `<rect>`, `<circle>`, or `<ellipse>` for animated elements (iOS Safari bug)
- Do NOT use CSS `transform-origin` (browser inconsistencies with SVG — GSAP handles transforms)
