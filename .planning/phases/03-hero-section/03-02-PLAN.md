  <action>
Create `src/lib/components/hero/LogoCarousel.svelte` — a CSS-only infinite scrolling marquee for brand logos at the bottom edge of the hero.

**Design (from CONTEXT.md + RESEARCH.md):**
- Sits at the bottom edge of the hero section (positioned by parent, not this component)
- CSS-only animation — zero JavaScript
- Slow, subtle scroll (30s+ full cycle)
- Falls back to static evenly-spaced row on `prefers-reduced-motion: reduce`
- Uses mask-image gradient for smooth fade-in/out at edges

**Placeholder logos (from Research Pitfall 7):**
Since real client logos are TBD, create 6 placeholder brand marks using small inline SVG geometric shapes:
1. A simple circle with a line through it
2. Two overlapping triangles
3. A square with inner square offset
4. A stylized "S" curve
5. Three parallel lines (hamburger-like)
6. A diamond shape

Each placeholder should be ~40px tall, muted color (`--color-text` with low opacity ~0.3), monochrome. These are meant to suggest "brands work with us" without being specific.

**CSS Marquee implementation (from Research):**
Use the Smashing Magazine CSS-only marquee pattern:

```css
.logo-carousel {
  --item-width: 120px;
  --item-height: 40px;
  --duration: 35s;
  --items: 6;
  --gap: 3rem;
}
```

Key technique:
- Items positioned `absolute` with `inset-inline-start` starting off-screen
- Each item gets staggered `animation-delay` via CSS custom property `--item-index`
- Single `@keyframes marquee` moves items from right to left
- `mask-image: linear-gradient(to right, transparent, black 10%, black 90%, transparent)` for smooth edge fade
- Duplicate the items in markup (render 12 items total — 2 copies of 6) to ensure seamless infinite loop

**Reduced motion:**
```css
@media (prefers-reduced-motion: reduce) {
  .logo-carousel__item {
    animation-play-state: paused;
    position: static;
    transform: none;
  }
  .logo-carousel {
    justify-content: space-evenly;
    mask-image: none;
    gap: var(--gap);
  }
}
```

**Component props:**
```svelte
<script lang="ts">
  interface Props {
    class?: string;
  }
  let { class: className = '' }: Props = $props();
</script>
```

**Initial state for animation:**
- Set the carousel wrapper to `opacity: 0` via CSS with class `.hero-carousel` for GSAP targeting. The orchestrator will fade it in as the last step of the entrance sequence.

**Important:**
- NO JavaScript animation — pure CSS @keyframes
- Assign `--item-index` via inline style on each item element (e.g., `style="--item-index: 0"`, `style="--item-index: 1"`, etc.)
- Use `display: flex` and `overflow-x: hidden` on the container
- Use `position: relative` on container with a set `height`

**Label (optional):**
Consider adding a small label above or to the left: "Trusted by" or "Working with" in `--text-xs`, `--font-sans`, `--color-text-muted` with `opacity: 0.5`. This provides context for the marks.
